    • We ask you now to guess how many points these people contributed to the collective pot in this task.


    For this question, you receive 100 tokens. Every token represents one study participant that, like you, lives in the UK. Please indicate for every possible contribution between 0 and 100 how many of the UK participants you believe chose that contribution.


    To answer, you can use the table below that contains every possible contribution between 0 and 10. Write in each cell the number of people you think chose to contribute that many tokens. Thus be careful, you have eleven numbers to guess and the numbers you write must sum up to 100.


    The final columns of the table show the sum of tokens assigned and the number of tokens left to assign. Below the table a figure shows how you distributed the tokens over the different contribution levels.
