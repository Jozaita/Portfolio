Ejemplo 1

En la ronda 1 se le ha agrupado al azar con otras {{ other_players_per_group }} personas. Cada uno de ustedes recibe {{ Constants.endowment }}. Usted contribuye {{ no_spending }} y los otros {{ Constants.other_players_per_group }} contribuyen {{ half_spending }} cada uno. De esta manera el total reunido por el grupo es {{ sum1 }} (= {{ no_spending }} + {{ other_players_per_group }} × {{ half_spending }}), que no llega a los {{ Constants.disaster_threshold }} necesarios.

Puesto que su grupo no logró acumular suficientes puntos, el ordenador selecciona aleatoriamente un número del 1 al 10. El ordenador elige 2 lo que significa que todos ustedes terminan la ronda con {{ Constants.disaster_low }} puntos.

Ejemplo 2

En la ronda 2 se le ha agrupado al azar con otras {{ other_players_per_group }} personas. Usted contribuye {{ three_quarters_spending }} y los otros {{ Constants.other_players_per_group }} contribuyen {{ three_quarters_spending }} cada uno. De esta manera el total reunido por el grupo es {{ sum2 }} (= {{ three_quarters_spending }} + {{ other_players_per_group }} × {{ three_quarters_spending}}), que está por encima de los {{ Constants.disaster_threshold }} necesarios.

Por lo tanto, todos ustedes guardan lo que no contribuyeron. Usted gana {{ one_quarter_spending }} puesto que contribuyó {{ three_quarters_spending }} (= {{ Constants.endowment }} - {{ three_quarters_spending }}) y de manera similar las otras personas ganan {{ one_quarter_spending }} puesto que contribuyeron {{ three_quarters_spending }}. (= {{ Constants.endowment }} - {{ three_quarters_spending }}).

Ejemplo 3

En la ronda 3 se le ha agrupado al azar con otras {{ other_players_per_group }} personas y, como en el ejemplo 1, usted contribuye {{ no_spending }} y los otros {{ other_players_per_group }} contribuyen {{ half_spending }} cada uno. De esta manera el total reunido por el grupo es {{ sum1 }} (= {{ no_spending }} + {{ other_players_per_group }} × {{ half_spending }}), que no llega a los {{ Constants.disaster_threshold }} necesarios.

Puesto que su grupo no logró acumular suficientes puntos, el ordenador selecciona aleatoriamente un número del 1 al 10. El ordenador elige 10 lo que significa que todos ustedes ganan los puntos que no contribuyeron. En este caso, usted gana {{ Constants.endowment|floatformat:"0" }}.
